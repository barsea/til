# GitHub上でディレクトリを作成する方法
1. 「Create new file」
2. ファイル名のフィールドで「github/.gitkeep」といったように、`作成したいディレクトリ名/.gitkeep`とする
3. コミットする

# GitHub上でファイルを別のディレクトリに移動する方法
1. 移動するファイルを見つける
2. 右上にあるエディットボタンをクリック
3. ファイル名のフィールドでファイル名の前に`移動したいディレクトリ名/`を挿入する
4. コミットする
